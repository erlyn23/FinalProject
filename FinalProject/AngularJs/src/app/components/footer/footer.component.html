Sistema Médico

Este sistema tiene el objetivo de gestionar un sistema médico, permitiendo al usuario añadir médicos, pacientes y habitaciones y gestionar ingresos, altas médicas y agendar citas, con una alta comodidad en el diseño y facilidad para usarlo.

Módulos
  • Administrar Médicos
  • Administrar Pacientes
  • Administrar Habitaciones
  • Agendar Citas
  • Registrar Ingresos
  • Dar Altas Médicas
